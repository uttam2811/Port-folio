# Uttam M — Portfolio

> Professional Black & Gold portfolio — military HUD aesthetic with GPU-accelerated animations

## Features
- **Greeting splash** — Hello · Hallo · Salut · வணக்கம் · नमस्ते · Ciao after loader
- GPU-accelerated animations via `transform` + `will-change` (zero layout/paint thrash)
- Black `#080808` + Gold `#c9a84c` professional color theme
- Cormorant Garamond (display) + DM Sans (body) + JetBrains Mono (mono)
- Smooth easeOutExpo progress bar
- Typewriter hero with multiple rotating subtitles
- Horizontal drag-to-scroll Evidence Board with modal decrypt
- Scroll-reveal on Profile/Skills/Certs
- Noise texture overlay for film grain depth
- HUD corners: live clock, coordinates, REC indicator
- Responsive + mobile hamburger

## Stack
React 18 · Vite 5 · Vanilla CSS · Google Fonts

## Run locally
```bash
npm install && npm run dev
```

## Deploy
Vercel → import `uttam2811/Port-folio` → Framework: Vite → Deploy
